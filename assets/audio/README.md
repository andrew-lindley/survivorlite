# Audio Assets

Place your sound effect files in this folder. The game expects the following audio files (MP3 format recommended):

## Weapon Sounds
- `laser.mp3` - Ship laser firing
- `fireball.mp3` - Fireball weapon
- `lightning.mp3` - Lightning aura zap
- `frost.mp3` - Frost nova burst
- `void.mp3` - Void zone activation
- `spirit.mp3` - Spirit orbs (optional, currently unused)

## Combat Sounds
- `hit.mp3` - Generic hit sound
- `enemy_death.mp3` - Enemy destruction
- `player_hit.mp3` - Player taking damage

## Pickup Sounds
- `xp_pickup.mp3` - Collecting XP gems
- `level_up.mp3` - Level up fanfare

## UI Sounds
- `button.mp3` - Button click (optional)
- `select.mp3` - Menu selection (weapon choice, etc.)

## Game Events
- `wave_start.mp3` - New wave beginning
- `victory.mp3` - Level victory
- `game_over.mp3` - Game over

## Notes
- All sounds are optional - the game will work without them
- Recommended format: MP3 (best browser compatibility)
- Also supported: OGG, WAV, M4A
- Keep files small (< 100KB each) for fast loading
- Sounds will automatically play when their trigger events occur
